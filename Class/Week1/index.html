GitHub and Atom

Why GitHub?

GitHub is a website which holds repositories. GitHub allows you to sync files between files and computers via the internet. A GitHub account and downloading the desktop version of GitHub allows you to keep up-to-date projects.

Benefits

  • Easily updated
  • Record of changes
  • Access from any computer

Using Atom and GitHub

Atom allows you to write and edit a variety of code. By saving an HTML file, for example, in the GitHub folder, you are able to sync the file with your GitHub account, and see them online. Atom's folder structure allows you to save your code directly to the GitHub folder on your computer.

The Process

First you must make a GitHub account. This can be created at github.com . After verifying your email address create your first repository. This repository must be named with your username and ".github.io" . Ex. username.github.io

The next step is downloading and installing the desktop version of GitHub. It can be found at desktop.github.com. You want to make a local copy of the repository that you set up online. Open GitHub desktop, and make sure you have an internet connection. Click the add button (+) on the top left of the screen. Save this folder, remembering where you want it kept.

Create the document you want to upload, and save it in the local repository folder that you created (through Atom you can save directly, and the folder structure will show in Atom). In GitHub desktop the new file should appear as a change. In order to sync the change you add a summary and a description if you want. Next you hit "Commit to master" at the bottom. These are your local changes. To make the changes online, hit the sync button in the top right corner, and wait for it to finish syncing.

Your changes are now on GitHub online!

You do not need to sync to GitHub everytime you make a change, but be sure to do it every so often.
